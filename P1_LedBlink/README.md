# Descripción
Control de un LED físico mediante los pines GPIO de la Raspberry Pi 4B utilizando Python. En esta práctica se configuró un circuito básico para realizar un parpadeo intermitente (blink).

# Configuración de Hardware
- **Pin de señal:** Pin físico 12 (GPIO 18 en modo BCM).
- **Resistencia:** 220Ω conectada a GND.
- **LED:** Rojo
